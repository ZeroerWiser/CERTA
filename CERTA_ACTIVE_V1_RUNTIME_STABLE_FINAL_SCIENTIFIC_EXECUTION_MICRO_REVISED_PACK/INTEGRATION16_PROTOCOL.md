# Integration16 Protocol

Integration16 is a no-gold real-table sentinel over the frozen first 16 dev samples and all three matched arms. It must close 16 samples and 48 arm identities, including deterministic no-call records.

Preserve Role V3 records, graph/group/card identities, retrieval references, raw Planner requests/responses, full and transport schemas, normalized payloads, all grounding assignments, closure, executed derivations, projection, provenance, registry, row failures and cost.

After close, `IntegrationFailureTaxonomist` reports the first failure stage per sample and arm. A terminal string without this taxonomy is incomplete. Only the unchanged sentinel PASS authorizes dev64.
