# Read-only Scientific Sub-agents

The main Codex agent alone edits source, calls endpoints, advances stages and writes terminals.

## GroundingAuthoritySemanticsAuditor

Reads all 192 grounding records, closure states and derivation bindings. Determines whether each multiplicity is a finite set of individually unique assignments or an internally ambiguous resolver result. Produces row-level evidence and no code.

## GateCReplayAuditor

Independently reconstructs old and shadow reconciliation and all Gate C terms from raw artifacts. Verifies unchanged thresholds and identifies the exact rows/tables/signatures responsible for any gain.

## HostileLeakageAndHeuristicAuditor

Searches the proposed correction for gold use, answer-correctness dependence, lexical ranking, first-match, arbitrary thresholds, benchmark-specific sample IDs and hidden truncation. Any finding blocks implementation.

## PaperClaimBoundaryAuditor

After Gate Q and after final dev/holdout results, maps artifacts to claims: executable contrast, selected-final gain, harmful revisions and cross-backbone consistency. It must require claim contraction when evidence is absent.

All agents are stage-bound, read-only, non-background and may not call model endpoints.
